# Frontend

Next.js app for the spending visualiser.

## Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Recharts

## Setup

1. Copy `.env.example` to `.env.local` and fill in values

## Scripts

| Command | Description |
|---|---|
| `pnpm dev` | Start the dev server |
| `pnpm build` | Build for production |
| `pnpm start` | Run the production build |
